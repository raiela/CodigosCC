A ideia do projeto está explicada no arquivo 'app-description'.

É preciso ter instalado em uma venv ou no computador os seguintes pacotes:
pip install flask
pip install flask-httpauth
pip install cast
pip install Flask-UUID

O projeto apresenta um serviço REST em Python que permite interagir com uma lista de nomes armazenada em memória O serviço REST permite:
Listar todos os nomes cadastrado na lista, consultar um unico nome, alterar um dado e excluir.
